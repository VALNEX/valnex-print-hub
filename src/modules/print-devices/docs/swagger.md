# Swagger - PrintDevices

## Base path

- /print-devices

## Endpoints

- POST /print-devices: create
- GET /print-devices: list with filters
- GET /print-devices/{id}: detail
- PATCH /print-devices/{id}: update
- DELETE /print-devices/{id}: remove

## DTOs

- CreatePrintDeviceDto
- UpdatePrintDeviceDto
- PrintDeviceDto
- FilterPrintDeviceDto
